\# Hive Glossary



This glossary defines common terms used throughout the Hive platform to help new contributors and users understand the system more easily.



---



\## Agent

An autonomous entity that performs tasks, makes decisions, and evolves over time based on feedback and system state.



\## Task

A discrete unit of work assigned to an agent. Tasks can be simple actions or complex multi-step processes.



\## Workflow

A structured sequence of tasks executed by one or more agents to achieve a business objective.



\## Hive

The core platform that orchestrates agents, tasks, and workflows, enabling self-evolving AI-driven automation.



\## Self-Evolving

The capability of agents to improve behavior or decision-making using feedback, learning, or state updates without manual intervention.
