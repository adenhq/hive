# Example Agent

## Overview
This example demonstrates a minimal Hive agent that can be executed from the repository root.
It is intended to help new contributors understand how agents are structured and run.

## Prerequisites
- Python 3.11+
- No API keys required
- No environment variables required

## Setup (from repository root)

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -e .
