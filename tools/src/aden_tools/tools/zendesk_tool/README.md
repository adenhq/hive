# Zendesk Tool

This tool allows Hive agents to interact with Zendesk tickets and search.

## Features
- Search tickets by query, status, and priority.
- Get detailed ticket information.
- Update ticket status, assignee, and add comments (internal or public).
- Health check to verify API connectivity.

## Configuration
Requires the following credentials:
- `ZENDESK_SUBDOMAIN`: Your Zendesk instance subdomain.
- `ZENDESK_EMAIL`: Your account email.
- `ZENDESK_API_TOKEN`: Your API token.

Refer to `docs/ZENDESK_INTEGRATION.md` for full documentation and reference agents.
